# Doctor Assistant
Stacks :
 - HTML
 - CSS
 - Javascript (Jquery)
 - PHP
 - MYSQL
 
 Features :
 - **ผู้ใช้**
	 - รายการยาของผู้ป่วย วิธีการรับประทานและรายละเอียดของตัวยา
	 - เมนูแผนที่เพื่อหาร้านยาใกล้เคียง
	 - เมนูแจ้งเตือนเมื่อถึงวันที่หมอนัดหมาย
- **โรงพยาบาล**
	- เพิ่มข้อมูลของผู้ป่วย
	- ดูอาการของผู้ป่วย
	- เพิ่มยา
	- จัดยาให้แก่คนไข้
	- นัดผู้ป่วย
- **ร้านขายยา**
	- แสดงยาของผู้ป่วยเพื่อใช้ในการจ่ายยา
	- ตรวจอาการของผู้ป่วย
